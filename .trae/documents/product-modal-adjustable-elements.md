# 产品详情框可调整元素清单

## 1. 弹窗容器和布局

### 1.1 弹窗背景 (modal-overlay)
- **背景颜色**: 可调整背景色和透明度
- **覆盖范围**: 可调整覆盖整个屏幕的方式

### 1.2 弹窗内容容器 (modal-content)
- **宽度**: 可调整固定宽度或百分比宽度
- **高度**: 可调整固定高度或最大高度
- **最大宽度**: 可调整不同屏幕尺寸下的最大宽度
- **最大高度**: 可调整不同屏幕尺寸下的最大高度
- **背景颜色**: 可调整背景色
- **边框半径**: 可调整圆角大小
- **阴影**: 可调整阴影效果
- **位置**: 可调整弹窗在屏幕中的位置
- **内边距**: 可调整内容与边框的间距

### 1.3 弹窗主体 (modal-body)
- **布局方式**: 可调整为 flex 或其他布局
- **间距**: 可调整左右区域的间距
- **响应式设计**: 可调整不同屏幕尺寸下的布局

### 1.4 左侧图片区域 (modal-images)
- **宽度**: 可调整在弹窗中的占比
- **边距**: 可调整与右侧信息区域的间距

### 1.5 右侧信息区域 (modal-info)
- **宽度**: 可调整在弹窗中的占比
- **边距**: 可调整与左侧图片区域的间距

## 2. 左侧图片区域

### 2.1 主图片 (modal-main-image)
- **宽度**: 可调整为百分比宽度
- **高度**: 可调整为固定高度或最大高度
- **最大高度**: 可调整为视口高度的百分比
- **图片显示方式**: 可调整为 object-contain, object-cover 等
- **边框半径**: 可调整圆角大小
- **光标样式**: 可调整为 pointer 或其他

### 2.2 主图片容器 (main-image-container)
- **边距**: 可调整与下方画廊的间距

### 2.3 放大提示文字 (zoom-hint)
- **字体大小**: 可调整文字大小
- **字体颜色**: 可调整文字颜色
- **边距**: 可调整与主图片的间距

### 2.4 图片画廊容器 (modal-gallery-container)
- **位置**: 可调整为相对定位或其他

### 2.5 画廊导航按钮 (gallery-nav-left, gallery-nav-right)
- **宽度**: 可调整按钮宽度
- **高度**: 可调整按钮高度
- **背景颜色**: 可调整背景色和透明度
- **边框半径**: 可调整圆角大小
- **阴影**: 可调整阴影效果
- **位置**: 可调整按钮位置
- **悬停效果**: 可调整悬停时的样式

### 2.6 画廊 (modal-gallery)
- **布局方式**: 可调整为 flex 或其他布局
- **间距**: 可调整图片之间的间距
- **滚动方式**: 可调整为水平滚动或其他

### 2.7 画廊图片项 (modal-gallery-item)
- **宽度**: 可调整每个图片项的宽度
- **高度**: 可调整每个图片项的高度

### 2.8 画廊缩略图 (gallery-thumb)
- **宽度**: 可调整为百分比宽度
- **高度**: 可调整为百分比高度
- **图片显示方式**: 可调整为 object-contain, object-cover 等
- **边框半径**: 可调整圆角大小
- **光标样式**: 可调整为 pointer 或其他
- **悬停效果**: 可调整悬停时的样式

### 2.9 画廊缩略图标签 (gallery-thumb-label)
- **字体大小**: 可调整文字大小
- **字体颜色**: 可调整文字颜色
- **边距**: 可调整与缩略图的间距
